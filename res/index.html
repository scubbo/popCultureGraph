What?

What do I do?

Drag Mode lets you click-and-drag any node to move it around.

Spread Mode lets you click a node to generate more links from it (this may take a few seconds if you're exploring connections no-one has explored before). You can click the same node more than once for more links!

Delete Mode lets you delete a node, if things are getting too crowded.

Prune Mode does the same as Delete Mode, but also submits a request to completely remove the node from the database. Use this for boring titles like talk shows, clip shows, and the like. (If you're technically-inclined, you can also submit a Pull Request to change this class).
